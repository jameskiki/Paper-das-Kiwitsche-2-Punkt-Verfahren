## Purpose

This is a scientific publication repository. The goal is to eventually produce a
peer-reviewable paper on the *Kiwitsche 2-Punkt-Verfahren* — a systematic
procedure for estimating optimal PID control parameters for complex industrial
plants (e.g., cooling systems) using Bang-Bang (two-point) control as the
initial excitation signal.

## Scientific Background

### Bang-Bang Control

A two-point (Bang-Bang) controller switches the manipulated variable between
two fixed levels (e.g., fully ON and fully OFF). For a stable plant with
sufficient gain and phase delay, this produces a *limit-cycle oscillation*
around the setpoint.

### Key Measured Quantities

| Symbol | Meaning |
|--------|---------|
| `T_u` | Oscillation period of the limit cycle [s] |
| `A_u` | Peak-to-peak amplitude of the controlled variable |
| `d`   | Switching hysteresis of the Bang-Bang controller |
| `M`   | Relay amplitude = (U_max - U_min) / 2 |

### FOPDT Identification (Transient Slope Method)

The KZV uses the step-response segments embedded within each limit-cycle
half-cycle to identify all three FOPDT parameters:

1. **Dead time L** — sign-change of `dy/dt` after a rising switch occurs at `t_sw + L`
2. **Time constant T** — at `t_sw + L` after a falling switch (with `U_min=0`):
   `T = y_L / |dy_L/dt|`
3. **Gain K** — from the rising switch at `t_sw + L`:
   `K = (T * slope_up + y_L) / U_max`

This is implemented in `analysis/parameter_estimation.py::identify_fopdt_from_transients`.

### PID Synthesis

Uses IMC-based formulas applied to the FOPDT model:
```
K_p = T / (K * (λ + L))
T_i = T
T_d = L / 2
```
Target closed-loop time constant `λ = L` (default).
